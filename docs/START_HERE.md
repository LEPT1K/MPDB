# 🚀 MPDB — БЫСТРЫЙ СТАРТ

**MPDB** (MITRE_PARSER_DATABASES) — парсер и обработчик баз данных MITRE (CAPEC, CWE, CVE, ATT&CK).

## Запуск графического интерфейса

### 1. Установите зависимости (если еще не установлены)
```bash
pip install -r requirements.txt
```

### 2. Запустите GUI
```bash
python gui/run_gui.py
```

### 3. Откройте браузер
Перейдите по адресу: **http://localhost:5000** (или http://localhost:5001 если 5000 занят)

## Порядок обработки данных

1. **Парсинг** — скачивание и разбор баз MITRE
2. **Связывание** — перекрёстные ссылки между базами
3. **Перевод** — многопоточный перевод полей на русский (Google Translate)
4. **Автозаполнение** — заполнение пустых полей (связи, факты из описаний, detection из STIX, шаблоны); выполняется после перевода

## Альтернативные варианты

### Консольное меню
```bash
python src/menu.py
```

### Прямой запуск всех этапов
```bash
python src/run_all.py
```

## ⚠️ Возможные проблемы

### Порт 5000 занят
GUI автоматически попробует порт 5001. Если оба заняты, закройте другие приложения или измените порт в `gui/run_gui.py`.

### Ошибки при импорте
Убедитесь, что все зависимости установлены:
```bash
pip install -r requirements.txt
```

### Ошибки при запуске процессов
Проверьте, что основной проект настроен корректно:
```bash
python src/main.py
```
